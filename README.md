# ASUS ROG Raikiri II Gamepad Viewer Skin

A custom CSS skin for [Gamepad Viewer](https://gamepadviewer.com/) designed to represent the ASUS ROG Raikiri II Xbox Wireless Controller.

## Features

- **Premium Design**: Sleek black finish with textured grip areas
- **RGB Lighting Effects**: Dynamic player indicator lighting
- **ROG Branding**: Signature ROG logo with pink/magenta accent colors
- **Full Button Support**: All standard Xbox controller buttons including:
  - A, B, X, Y face buttons with authentic colors
  - D-Pad (Up, Down, Left, Right)
  - L1/R1 bumpers and L2/R2 triggers
  - Analog sticks (L3/R3)
  - View and Menu buttons
  - Xbox logo button
  - Optional: Paddle buttons support (M1-M4)
- **Visual Feedback**: All buttons light up when pressed with the signature ROG pink/magenta glow
- **Disconnected State**: Semi-transparent with grayscale filter when controller disconnects

## Usage

### Direct Link
Use this direct CSS link in Gamepad Viewer:

```
https://raw.githubusercontent.com/[YOUR-USERNAME]/ghost-raikiri/main/raikiri-ii.css
```

### In Gamepad Viewer
1. Go to [Gamepad Viewer](https://gamepadviewer.com/)
2. Connect your controller
3. Click the settings/gear icon
4. Paste the CSS URL into the "Custom CSS" field
5. Click "Apply"

### Example URL
```
https://gamepadviewer.com/?p=1&css=https://raw.githubusercontent.com/[YOUR-USERNAME]/ghost-raikiri/main/raikiri-ii.css
```

## Customization

You can easily customize the colors by editing the CSS file:

- **ROG Pink/Magenta**: `#ff0066`, `#cc0052`, `#990040`
- **Background**: `#1a1a1a`, `#2d2d2d`
- **Button Colors**:
  - A (Green): `#4ade80`, `#22c55e`
  - B (Red): `#ff4444`, `#cc0000`
  - X (Blue): `#60a5fa`, `#3b82f6`
  - Y (Yellow): `#fbbf24`, `#f59e0b`

## Controller Information

The ASUS ROG Raikiri II is a premium Xbox-licensed wireless controller featuring:
- Tri-mode connectivity (2.4GHz, Bluetooth, Wired)
- 4 programmable back buttons
- ESS DAC for high-quality audio
- AURA RGB lighting
- Adjustable joystick sensitivity
